# 缩写

将短语转换为，其首字母缩写词。

技术人员都喜欢他们的 TLA(三字母缩略语(Three Letter Acronyms)显得高大上)!

通过编写将诸如 Portable Network Graphics 之类的长名称，转换为其首字母缩略词(PNG)的程序,帮助生成一些术语。
