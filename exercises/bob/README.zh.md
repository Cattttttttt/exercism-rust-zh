# 鲍勃-bob

鲍伯是一个懒散的青少年.在谈话中,他的反应非常有限.

- 鲍伯回答:"Sure."，如果你问他一个问题.

- 他回答:"Whoa, chill out!"，如果你对他大喊大叫.

- 他回答"Calm down, I know what I'm doing!"，如果你大声问他问题.

- 他说"Fine. Be that way!"，如果你喊他,而不说任何话.

- 他回答"Whatever"，给剩下的对话

鲍勃的对话伙伴，在书面交流方面是一个纯粹主义者,并且总是遵循关于 *英语句子标点* 的通用规则.
