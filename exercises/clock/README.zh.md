# 时钟

实现一个处理没有日期的时间的时钟.

您应该可以添加和减去分钟数.

代表相同时间的两个时钟应该彼此相等.

## .to_string()的 Rust 特征

你为 Clock 结构实现了.to_string()吗?

如果是这样,请尝试实施[Display trait](https://doc.rust-lang.org/std/fmt/trait.Display.html)代替时钟.

特征允许实现各种类型的功能的通用方法.

对于其他学习,如何为 Clock 类型实现 String :: from?
