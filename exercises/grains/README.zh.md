# 谷物

计算棋盘上的小麦粒数,假设每个方格的数量增加一倍.

曾经有一位睿智的仆人拯救了王子的生命。国王承诺支付仆人梦寐以求的一切。知道国王喜欢国际象棋,仆人告诉国王他想吃小麦粒，在棋盘的第一个正方形上放一粒小麦。而接下来的方格是两粒.四粒小麦放在第三格,依此类推.

棋盘上有 64 个方格.

编写代码，用来显示:

- 每个方格上有多少谷物,和
- 谷物总数
