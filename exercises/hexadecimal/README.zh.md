# 十六进制的

使用第一原理将十六进制数(表示为字符串)(例如"10af8c")转换为其十进制等效数(即,不可以使用内置或外部库来完成转换).

在Web上,我们使用十六进制表示颜色,例如Green:008000,TEAL:008080,海军:000080).

程序应该处理无效十六进制字符串.
