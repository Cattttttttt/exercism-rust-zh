# 十六进制

使用第一原理，将用字符串表示的十六进制数，(例如"10af8c")，转换为其十进制等效数(即，不可以使用内置或外部库来完成转换)。

在 Web 上，我们使用十六进制表示颜色，例如( green: 008000,
teal: 008080, navy: 000080)。

程序应该处理无效十六进制字符串。
