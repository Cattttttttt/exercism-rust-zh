# 等值线

确定是否一词,或短语是等值线(`isograms`).

在等值线(又称为"无定形的 Word 是一个字或短语")没有重复字母的单词或短语的逻辑术语，但允许连字符和空格出现多次。

等值线的例子:

- lumberjacks
- background
- downstream
- six-year-old

检测 **isograms** 单词，但它不是等值线,因为`s`重复了.
