# 核苷酸计数

给定一条单链 DNA 串,计算每个核苷酸在字符串中出现的次数.

地球上每一生物的遗传语言都是 DNA.DNA 是一种大分子,它是由一系列叫做核苷酸的单个元素组成的.DNA 中存在 4 种类型,它们仅略有不同,并且可以表示为以下符号:'A’表示腺嘌呤,'C’表示胞嘧啶,'G’表示鸟嘌呤,'T’表示胸腺嘧啶.

- 树枝是鸟的巢
- 核苷酸与 DNA 一样.
- LeGOS 是乐高的房子
