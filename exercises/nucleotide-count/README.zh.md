# 核苷酸(Nucleotide)计数

给定一条单 DNA 链 ,计算每个核苷酸在字符串中出现的次数.

地球上每一生物的遗传语言都是 DNA.DNA 是一种大分子,它是由一系列叫做核苷酸的单个元素组成。DNA 中存在 4 种类型,它们仅略有不同,并且可用以下符号表示:'A'表示腺嘌呤,'C'表示胞嘧啶,'G'表示鸟嘌呤,'T'表示胸腺嘧啶.
