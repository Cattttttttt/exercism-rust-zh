# 全字母句（pangram）

判断句子是否是全字母句。全字母句(希腊语:παγρμμα,pan 语法,"每个字母")是一个使用字母表中每个字母，至少一次的句子。最著名的英语是 全字母句:

> The quick brown fox jumps over the lazy dog.

字母表由 ASCII 字母`a`到`z`的全部组成，并且不区分大小写。输入不能包含非 ASCII 符号.
