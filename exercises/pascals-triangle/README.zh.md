# 帕斯卡三角

计算 Pascal 的三角形直到给定的行数.

在 Pascal 三角形中,通过将数字添加到前一行中当前位置的左右来计算每个数字.

```text
    1
   1 1
  1 2 1
 1 3 3 1
1 4 6 4 1
# ... etc
```
