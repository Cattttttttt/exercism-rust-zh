# 杨辉三角形

计算给定的行数的 杨辉三角形。

在 杨辉 三角形中, 可以看到，一个数由它头顶左右两个数相加得到的。

```text
    1
   1 1
  1 2 1
 1 3 3 1
1 4 6 4 1
# ... etc
```
