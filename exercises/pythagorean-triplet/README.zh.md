# 勾股数

勾股数是一组,三个自然数,{a,b,c}, 如

```text
a**2 + b**2 = c**2
```

这样,

```text
a < b < c
```

例如,

```text
3**2 + 4**2 = 9 + 16 = 25 = 5**2.
```

问: 请找出 **a，b，c**，恰好符合勾股定理, 而其中 a+b+c=1000.

返回， `a * b * c` 值.
