# 毕达哥拉斯三胞胎

毕达哥拉斯三胞胎是一组三个自然数,{a,b,c},

```text
a**2 + b**2 = c**2
```

这样,

```text
a < b < c
```

例如,

```text
3**2 + 4**2 = 9 + 16 = 25 = 5**2.
```

恰好有一个毕达哥拉斯三重态,其中 a+b+c=1000.

找到产品 A*乙* c.
