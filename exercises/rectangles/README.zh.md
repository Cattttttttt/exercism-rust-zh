# 矩形

计数 ASCII 图中的矩形,如下所示.

```text
   +--+
  ++  |
+-++--+
|  |  |
+--+--+
```

上面的图表包含 6 个矩形:

```text
+-----+
|     |
+-----+
```

```text
   +--+
   |  |
   |  |
   |  |
   +--+
```

```text
   +--+
   |  |
   +--+
```

```text
   +--+
   |  |
   +--+
```

```text
+--+
|  |
+--+
```

```text
  ++
  ++
```

你可以假设输入总是一个适当的矩形(即每行的长度等于第一行的长度).
