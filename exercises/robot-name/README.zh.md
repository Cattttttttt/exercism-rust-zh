# 机器人名称

管理机器人工厂设置.

当机器人从工厂车间出来时,他们没有名字.

第一次启动时,会生成一个随机名称,格式为两个大写字母后跟三个数字,如 RX837 或 BC811.

每隔一段时间我们就需要将机器人重置为出厂设置,这意味着他们的名字会被擦除.下次你问,它会以一个新的随机名称回复.

名称必须是随机的:它们不应遵循可预测的顺序.随机名称表示发生碰撞的风险.您的解决方案必须确保每个现有机器人都具有唯一名称.
