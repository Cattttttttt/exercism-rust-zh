# Two Fer

`Two-fer`或`2-fer`是 two for one 的缩写。One for you and one for me。

```text
"One for X, one for me."
```

当 X 是一个名字或`you`。

如果给定名称是`Alice`，结果应该是`One for Alice, one for me.`。如果没有给出名称，结果应该是`One for you, one for me.`。
